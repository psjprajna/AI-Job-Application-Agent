# AI Job Application Agent — Browser Extension

Manifest V3 extension that detects ATS application forms (Greenhouse, Lever, Ashby) and offers a single-click autofill from the latest application generated by the local app.

## Load it locally

1. Make sure the app is running at `http://localhost:3000`.
2. Open `chrome://extensions`.
3. Toggle **Developer mode** on.
4. Click **Load unpacked** → select this `extension/` directory.
5. Visit a Greenhouse / Lever / Ashby job posting (any company). When the apply form is detected you'll see an **⚡ Autofill with AI Job Agent** button in the bottom-right.
6. Click the extension icon to point it at a different API base URL.

## What it fills

The content script matches form labels against common patterns (`name`, `email`, `phone`, `linkedin`, `github`, `portfolio`, `cover letter`, …) and fills them from `GET /api/applications/latest` — which returns the most recently generated application + parsed resume.

## Files

- `manifest.json` — MV3 manifest, host permissions for the three ATS domains
- `background.js` — service worker, fetches the latest application from the app API
- `content.js` — detects fields, injects the floating autofill button, fills inputs
- `popup.html` / `popup.js` — settings popup (API base URL)
